# 🎬 IMDb Top 25 Movies Scraper

A simple Python tool to scrape the *Top 25 movies* from IMDb using BeautifulSoup. This project is beginner-friendly and includes a .exe file for users who don't have Python installed.

this is for educational purpose only and imdb content belongs to imdb only
---

## ⚡ Features

- Scrapes movie *titles* and *raanking*
- Outputs results to the terminal
- Includes a standalone .exe (Windows only)
- Clean, minimal codebase
